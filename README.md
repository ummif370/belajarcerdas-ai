# BelajarCerdas AI

## Deskripsi
BelajarCerdas AI merupakan aplikasi berbasis Artificial Intelligence (AI) yang membantu murid SMP memperoleh strategi belajar yang dipersonalisasi berdasarkan mata pelajaran, materi yang sulit dipahami, target nilai, dan waktu belajar.

## Fitur
- Analisis kebutuhan belajar
- Strategi belajar yang dipersonalisasi
- Metode belajar yang sesuai
- Jadwal belajar 7 hari
- Materi prioritas
- Rekomendasi kata kunci belajar
- Tips belajar
- Motivasi belajar
- Doa sebelum belajar

## Teknologi
- Python
- Streamlit
- Google Gemini API
- HTML & CSS (untuk tampilan)

## Model AI
Gemini 2.5 Flash

## Cara Menjalankan
1. Install library:
   ```
   pip install -r requirements.txt
   ```
2. Tambahkan file `.env` yang berisi:
   ```
   GEMINI_API_KEY=API_KEY_ANDA
   ```
3. Jalankan aplikasi:
   ```
   streamlit run app.py
   ```
